# VMware Workstation Pro 26H1

VMware Workstation Pro 是一款功能强大的桌面虚拟化软件，允许用户在单一物理机上同时运行多个操作系统。

## 版本信息

- **软件名称**: VMware Workstation Pro
- **版本**: 26H1 (Build 25388281)
- **文件名**: `VMware-Workstation-Full-26H1-25388281.exe`
- **文件大小**: ~287 MB

## 下载

请前往 [Releases 页面](../../releases) 下载最新版本。

## 系统要求

- Windows 10/11 64-bit 或 Linux 64-bit
- 至少 4GB 内存（推荐 8GB 以上）
- 至少 1.5GB 磁盘空间用于安装

## 安装说明

1. 从 Releases 页面下载安装包
2. 双击运行安装程序 (`VMware-Workstation-Full-26H1-25388281.exe`)
3. 按照安装向导完成安装
4. 安装完成后重启计算机

## 主要功能

- 在同一台电脑上运行多个操作系统
- 支持 Windows、Linux、macOS 等客户操作系统
- 快照和克隆功能
- 虚拟网络配置
- 3D 图形加速支持
